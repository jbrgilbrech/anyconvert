Native macOS file conversion

Convert images, documents, and audio without leaving your Mac.

AnyConvert ships as a terminal tool and a basic GUI app. It wraps native macOS conversion tools, so common tasks like JPG to PNG, TXT to RTF, SVG to PNG, MP3 to WAV, and more stay local.

View Repository Build It Locally 
$ anyconvert convert --input photo.jpg --to png
Converted photo.jpg -> /Users/you/Desktop/photo.png

$ anyconvert formats track.mp3
wav     WAV
aiff    AIFF
caf     CAF
m4a     M4A
flac    FLAC

Images

PNG, JPG, TIFF, GIF, BMP, HEIC, JP2, PDF, ICO, ICNS, TGA, PSD, AVIF, PBM.

Documents

TXT, RTF, RTFD, HTML, DOC, DOCX, ODT, WordML, Web Archive.

Audio

WAV, AIFF, AIFC, CAF, M4A, MP3, AAC, FLAC, Opus.

Terminal

Fast CLI workflow

swift run anyconvert formats /path/to/input.jpg
swift run anyconvert convert --input /path/to/input.jpg --to png
swift run anyconvert convert --input /path/to/input.mp3 --to wav

Desktop

Basic GUI app

Pick a file, choose an output format, and save the converted file. The app bundle is generated from the same shared Swift conversion engine as the CLI.

./scripts/build-app.sh
open dist/AnyConvert.app

Reality check

Not literally every file format
