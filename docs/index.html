Supported

          
Media-first coverage

          
Images like JPG, PNG, WebP, GIF, BMP, TIFF, and SVG. Audio like MP3, WAV, FLAC, OGG, AAC, and M4A. Video like MP4, WebM, MOV, MKV, and GIF output.

        


        

          
Limits

          
Not literally every format

          
Browser-only conversion is realistic for many media formats, not every proprietary document type. This version focuses on the formats a static client-side app can handle well.

        


        

          
Privacy

          
No upload required

          
Files stay in memory inside the browser tab unless you choose to download the converted result. There is no server-side processing in this GitHub Pages app.
